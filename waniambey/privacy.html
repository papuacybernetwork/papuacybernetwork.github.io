Kebijakan Privasi PaceMace Jayapura

Terakhir diperbarui: 14 April 2026

PENTING: Aplikasi PaceMace Jayapura adalah platform independen yang dikembangkan oleh PT. Papua Cyber Network. Aplikasi ini BUKAN aplikasi resmi pemerintah dan TIDAK mewakili entitas pemerintah manapun.

Aplikasi PaceMace Jayapura berkomitmen untuk melindungi privasi pengguna kami. Kebijakan ini menjelaskan bagaimana kami mengelola informasi dalam aplikasi kami.

1. Sumber Informasi Pemerintah

Segala informasi terkait layanan kependudukan dan catatan sipil yang diakses melalui aplikasi ini bersumber langsung dari situs web resmi Dinas Kependudukan dan Pencatatan Sipil Kota Jayapura di: http://jayapurakota.go.id/

2. Informasi yang Kami Kumpulkan

Aplikasi ini berfungsi sebagai media akses (mobile-optimized browser) menuju sistem web resmi pemerintah. Kami menerapkan kebijakan Zero-Data Retention:

  • Kami tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda (seperti NIK, nama, atau foto dokumen) ke server internal PT. Papua Cyber Network.
  • Semua data yang Anda masukkan diproses secara langsung oleh sistem tujuan (Dukcapil) melalui koneksi terenkripsi.

3. Penggunaan Data

Data yang Anda masukkan saat menggunakan layanan hanya digunakan untuk keperluan transaksi layanan kependudukan yang Anda pilih pada sistem resmi pemerintah terkait.

4. Keamanan Data

Meskipun kami tidak menyimpan data Anda, aplikasi kami memastikan transmisi data dilakukan melalui protokol aman (HTTPS/SSL) untuk mencegah intersepsi oleh pihak ketiga selama proses akses ke server pemerintah.

5. Tautan Pihak Ketiga

Aplikasi ini berisi tautan ke situs web pemerintah. Kami tidak bertanggung jawab atas kebijakan privasi atau konten dari situs web pihak ketiga tersebut. Kami menyarankan Anda untuk membaca kebijakan privasi pada situs web resmi pemerintah yang bersangkutan.
